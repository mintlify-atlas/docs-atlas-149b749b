---
title: "Get an Event"
description: "Retrieve a single event by its UUID, including all stored fields."
---

`GET /v1/events/:id`

**Auth:** Agent key (`events:read`) or JWT

Fetches the full event record for a given UUID. The event must belong to an
app that is accessible to your credentials.

## Path parameters

<ParamField path="id" type="string" required>
  The event UUID.
</ParamField>

## Response

<ResponseField name="id" type="string">Event UUID.</ResponseField>
<ResponseField name="app_id" type="string">App that received the event.</ResponseField>
<ResponseField name="session_id" type="string">Session UUID.</ResponseField>
<ResponseField name="user_id" type="string | null">User identifier, or `null` if none was provided.</ResponseField>
<ResponseField name="api_key_id" type="string | null">ID of the API key used to ingest this event.</ResponseField>
<ResponseField name="level" type="string">`info` | `debug` | `warn` | `error`</ResponseField>
<ResponseField name="message" type="string">Log message or event name.</ResponseField>
<ResponseField name="source_module" type="string | null">Module or component that emitted the event.</ResponseField>
<ResponseField name="screen_name" type="string | null">Active screen or route at the time of the event.</ResponseField>
<ResponseField name="environment" type="string | null">Runtime environment (`ios`, `android`, `web`, etc.).</ResponseField>
<ResponseField name="app_version" type="string | null">App version string.</ResponseField>
<ResponseField name="build_number" type="string | null">Build number.</ResponseField>
<ResponseField name="os_version" type="string | null">OS version string.</ResponseField>
<ResponseField name="device_model" type="string | null">Device model identifier.</ResponseField>
<ResponseField name="locale" type="string | null">BCP 47 locale tag.</ResponseField>
<ResponseField name="is_dev" type="boolean">Whether this was a development-build event.</ResponseField>
<ResponseField name="custom_attributes" type="object | null">Key-value map of custom attributes attached at ingest time.</ResponseField>
<ResponseField name="experiments" type="object | null">Experiment name → variant assignments at the time of the event.</ResponseField>
<ResponseField name="client_event_id" type="string | null">Client-assigned deduplication UUID, if supplied.</ResponseField>
<ResponseField name="timestamp" type="string">ISO 8601 timestamp of when the event occurred.</ResponseField>
<ResponseField name="received_at" type="string">ISO 8601 timestamp of when OwlMetry received and stored the event.</ResponseField>

## Example

```bash
curl https://api.yourdomain.com/v1/events/e1a2b3c4-0000-0000-0000-000000000101 \
  -H "Authorization: Bearer owl_agent_xxx"
```

## Errors

| Status | Meaning |
|--------|--------|
| `404`  | Event not found, or it belongs to an app your credentials cannot access. |
