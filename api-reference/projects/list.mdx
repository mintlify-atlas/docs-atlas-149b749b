---
title: "List & Retrieve Projects"
description: "List all projects your team can access, or fetch a single project with its apps."
---

## List projects

`GET /v1/projects`

**Auth:** Agent key (`projects:read`) or JWT

Returns all non-deleted projects belonging to the teams your credentials can access.

### Query parameters

<ParamField query="team_id" type="string">
  Scope the response to a single team. If you supply a `team_id` you are not a
  member of, the response will be an empty array rather than a 403.
</ParamField>

### Response

<ResponseField name="projects" type="array">
  List of project objects.
  <Expandable title="Project object">
    <ResponseField name="id" type="string">Unique project UUID.</ResponseField>
    <ResponseField name="team_id" type="string">UUID of the owning team.</ResponseField>
    <ResponseField name="name" type="string">Human-readable project name.</ResponseField>
    <ResponseField name="slug" type="string">
      URL-safe identifier. Lowercase letters, numbers, and hyphens only.
      Unique within the team.
    </ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 creation timestamp.</ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash
curl https://api.yourdomain.com/v1/projects \
  -H "Authorization: Bearer owl_agent_xxx"
```

```json
{
  "projects": [
    {
      "id": "a1b2c3d4-0000-0000-0000-000000000001",
      "team_id": "t1b2c3d4-0000-0000-0000-000000000099",
      "name": "Mobile Apps",
      "slug": "mobile-apps",
      "created_at": "2025-01-15T10:00:00.000Z"
    }
  ]
}
```

## Get a project

`GET /v1/projects/:id`

**Auth:** Agent key (`projects:read`) or JWT

Fetches a single project by its UUID and includes the list of active apps that belong to it.

### Path parameters

<ParamField path="id" type="string" required>
  The project UUID.
</ParamField>

### Response

<ResponseField name="id" type="string">Project UUID.</ResponseField>
<ResponseField name="team_id" type="string">Owning team UUID.</ResponseField>
<ResponseField name="name" type="string">Project name.</ResponseField>
<ResponseField name="slug" type="string">URL-safe identifier.</ResponseField>
<ResponseField name="created_at" type="string">ISO 8601 creation timestamp.</ResponseField>
<ResponseField name="apps" type="array">
  Apps belonging to this project.
  <Expandable title="App object">
    <ResponseField name="id" type="string">App UUID.</ResponseField>
    <ResponseField name="team_id" type="string">Owning team UUID.</ResponseField>
    <ResponseField name="project_id" type="string">Parent project UUID.</ResponseField>
    <ResponseField name="name" type="string">App name.</ResponseField>
    <ResponseField name="platform" type="string">`apple` | `android` | `web` | `backend`</ResponseField>
    <ResponseField name="bundle_id" type="string | null">Bundle or package identifier. `null` for backend apps.</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 creation timestamp.</ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash
curl https://api.yourdomain.com/v1/projects/a1b2c3d4-0000-0000-0000-000000000001 \
  -H "Authorization: Bearer owl_agent_xxx"
```

```json
{
  "id": "a1b2c3d4-0000-0000-0000-000000000001",
  "team_id": "t1b2c3d4-0000-0000-0000-000000000099",
  "name": "Mobile Apps",
  "slug": "mobile-apps",
  "created_at": "2025-01-15T10:00:00.000Z",
  "apps": [
    {
      "id": "a9b8c7d6-0000-0000-0000-000000000042",
      "team_id": "t1b2c3d4-0000-0000-0000-000000000099",
      "project_id": "a1b2c3d4-0000-0000-0000-000000000001",
      "name": "iOS App",
      "platform": "apple",
      "bundle_id": "com.example.myapp",
      "created_at": "2025-01-16T08:30:00.000Z"
    }
  ]
}
```

### Errors

| Status | Meaning |
|--------|--------|
| `404`  | Project not found or not accessible to your credentials. |
