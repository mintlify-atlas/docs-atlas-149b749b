---

This guide walks you through setting up OwlMetry locally, connecting the CLI, and sending your first events using the Node.js SDK.

## Prerequisites

- Node.js >= 20
- PostgreSQL >= 15
- pnpm

## 1. Install the CLI

```bash
npm install -g @owlmetry/cli
```

## 2. Start the server

If you're running OwlMetry locally:

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/Jasonvdb/owlmetry.git
    cd owlmetry
    pnpm install
    ```
  </Step>
  <Step title="Create the database">
    ```bash
    createdb owlmetry
    cp .env.example .env
    # Edit .env with your DATABASE_URL and JWT_SECRET
    ```
  </Step>
  <Step title="Run migrations and seed">
    ```bash
    pnpm db:migrate
    pnpm dev:seed
    ```

    <Note>
      `dev:seed` creates a demo user, team, project, app, and deterministic API keys for local development. Do **not** run this in production.
    </Note>
  </Step>
  <Step title="Start the API server">
    ```bash
    pnpm dev:server
    # Server runs on http://localhost:4000
    ```
  </Step>
</Steps>

## 3. Configure the CLI

```bash
owlmetry setup --endpoint http://localhost:4000 --api-key <your-agent-key>
```

This saves the endpoint and key to `~/.owlmetry/config.json`. Alternatively, use environment variables:

```bash
export OWLMETRY_ENDPOINT=http://localhost:4000
export OWLMETRY_API_KEY=<your-agent-key>
```

## 4. Verify the connection

```bash
owlmetry projects
```

You should see your projects listed. Try listing apps:

```bash
owlmetry apps
```

## 5. Install the Node.js SDK

In your application:

<CodeGroup>
```bash npm
npm install @owlmetry/node
```
```bash pnpm
pnpm add @owlmetry/node
```
```bash yarn
yarn add @owlmetry/node
```
</CodeGroup>

## 6. Send your first events

```typescript index.ts
import { Owl } from '@owlmetry/node';

Owl.configure({
  endpoint: 'http://localhost:4000',
  apiKey: 'owl_client_xxx', // client key from your app settings
  serviceName: 'my-server',
  appVersion: '1.0.0',
});

Owl.info('Server started');
Owl.info('User logged in', { route: '/auth/login', userId: 'user_123' });
Owl.error('Payment failed', { error: 'insufficient_funds', amount: '99.00' });

// Graceful shutdown (flushes buffered events)
await Owl.shutdown();
```

## 7. Query your events

```bash
owlmetry events --since 5m
```

You should see the events you just sent. Use `--format json` for machine-readable output:

```bash
owlmetry events --since 5m --format json
```
